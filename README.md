## Notebooks

The project is organized into three Jupyter notebooks:

- **1_data_cleaning_preparation.ipynb** – data loading, cleaning, and preprocessing.
- **2_eda.ipynb** – exploratory data analysis (EDA) including rating, price, genre, and text analysis.
- **3_data_modeling.ipynb** – modeling stage including clustering, sentiment classification, and model evaluation.
